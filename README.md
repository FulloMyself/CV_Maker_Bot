# EME ATS-Compliant CV Maker

An intelligent CV creation tool that generates ATS-friendly resumes while collecting required documents (Certified ID & Proof of Residence).

## 🚀 Features

- ✅ Generate professional, ATS-compliant CVs
- ✅ Upload and store certified ID and proof of residence
- ✅ Collect comprehensive user information
- ✅ Export CVs as PDF
- ✅ Admin dashboard to view all submissions
- ✅ Database storage for all user data

## 🛠️ Tech Stack

- **Backend**: Flask (Python)
- **Database**: SQLite
- **PDF Generation**: ReportLab
- **Frontend**: HTML5, CSS3, JavaScript

## 📋 Prerequisites

- Python 3.11 or higher
- pip (Python package manager)
